# Bikeshare
The present work is a data analysis of Bike Share Toronto Ridership Data.

## Important Notes
-   `data_preprocessing` files are not required to run, they are primarily useful if you are repeating this analysis with a different BIA.
-   `starter_code` files are designed to help you quickly access the data and other useful information prior to performing EDA.
-   `CA_` files correspond to comparative analysis scripts
-   `EDA_` files correspond to exploratory data analysis scripts
